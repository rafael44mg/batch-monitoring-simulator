# Batch Monitoring Simulator

Simulador de monitoramento batch inspirado em rotinas operacionais de equipes de Scheduling e Suporte N1.

## Objetivo
Simular análise de logs, detecção de incidentes e registro de eventos críticos em ambientes de TI.

## Funcionalidades
- Leitura de logs batch
- Detecção automática de erros e alertas
- Registro de incidentes em JSON
- Simulação de monitoramento operacional

## Tecnologias
- Python
- Processamento de logs
- Estrutura inspirada em operações de TI

## Execução
```bash
python monitor.py

## Fluxo Operacional

1. Leitura dos logs batch
2. Identificação de erros e alertas
3. Registro automático do incidente
4. Simulação de atuação do suporte N1

## Exemplo de execução

Incidentes detectados:
ERROR Job REPORT_GENERATION failed TIMEOUT
ALERT Disk usage above threshold

Incidente registrado com status OPEN.

## Cenário Simulado

Este projeto simula um ambiente onde rotinas batch são monitoradas
continuamente e incidentes são registrados para atuação do suporte N1,
seguindo práticas comuns em operações de TI corporativas.
