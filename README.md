# Batch Monitoring Simulator

Simulador de monitoramento batch inspirado em rotinas operacionais de equipes de Scheduling e Suporte N1.

## Objetivo
Simular análise de logs, detecção de incidentes e registro de eventos críticos em ambientes de TI.

## Funcionalidades
- Leitura de logs batch
- Detecção automática de erros e alertas
- Registro de incidentes em JSON
- Simulação de monitoramento operacional

## Tecnologias
- Python
- Processamento de logs
- Estrutura inspirada em operações de TI

## Execução
```bash
python monitor.py
